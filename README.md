# e-wallet-clone (개선판)

개인 연습용 전자지갑 자격증 클론 UI.

## 개선 포인트
- 데이터 기반 렌더링 (`app.js` 내부 `certData`)
- 검색/탭/정렬 동작
- 전면 / 후면 / 상세 화면 분리
- 발급 완료 후 `보관 -> 발급` 상태 이동
- 사용자 사진 반영
- 외부 라이브러리 없는 QR 스타일 패턴 생성
- 파일 하나 열면 바로 실행되는 순수 HTML/CSS/JS 구조

## 실행
브라우저에서 `index.html` 열기.

## 데이터 바꾸는 곳
`app.js` 상단의 `certData` 배열 수정.
